Today I Learned

2 Jan 2021

  • OPAQUE is a password authentication method that protects against data leaks
  • OPAQUE is a combination of an Oblivious Pseudorandom Function (OPRF) and an Authenticated Key Exchange (AKE)
  • An OPRF is a way for a client and server to jointly compute a Hash(password, salt) such that the client doesn't learn the salt and the server doesn't learn the password or its hash
